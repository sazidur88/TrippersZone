4.0 Class Design

4.1 Class Description

1.User: We are using user class to store all the data about a user. Everytime a user registers to our system a new row of data will be stored and also for sign in we will match data from this class.

2.Renter:The people who are going to visit highly tourist area then they feel need of living place. So then renter can easily access room using this application.

3.House Owner: House owner can easily rent their free rooms and houses to the renter then easily access to earn money from our application.

4.Room/House:Room/House class consisting of room rents ,details and other sepecfication of room then usually update their information.

5.Search for Room:Renter can usaully search their rooms using actual location and then find their desire place for living.
